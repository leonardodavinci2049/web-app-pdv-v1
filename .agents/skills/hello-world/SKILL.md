# Hello World

Use the Hello, World skill to respond to the user when they enter the phrase "hello world".

Respond with "Hello World" in ascii art.


## Workflow

1. Run the [script](./scripts/get-system-info.js) to obtain system information.

💡
2. Respond with the [template](./TEMPLATE.md)

3. Provide the system information obtained from the script.
